Static Site Generator Project

The Static Site Generator Project gave me the basic building blocks to create websites like this Tolkien Fan Page. I would like to take it a few steps further and, while doing so, I want to document any features I've added to it here. You'll also find a To Do list with features I might add later.

Done

  • Escaped delimiters are ignored by the `split_nodes_delimiters` function

Fixes

  • 2025-04-01 : Inside text_to_textnodes, delimiters were processed before images and links, which gave errors when a URL contained a delimiter like an underscore. Fixed by running split_nodes_image and split_nodes_link before the loop of split_nodes_delimiter.
